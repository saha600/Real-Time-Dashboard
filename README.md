## Project Description

This project is a Real-Time Student Performance Dashboard developed using Streamlit, Pandas, and Plotly. It provides an interactive dashboard to visualize student performance through KPI cards, charts, filters, and downloadable reports.

---

## Features

- Interactive Dashboard
- Sidebar Filters
- Search Student
- Download CSV
- KPI Cards
- Bar Chart
- Line Chart
- Pie Chart
- Top Performer Section
- Student Data Table
- Responsive Layout

---

## Technologies Used

- Python 3
- Streamlit
- Pandas
- Plotly

---

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
streamlit run app.py
```

or

```bash
python -m streamlit run app.py
```
